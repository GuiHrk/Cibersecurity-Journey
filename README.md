# 🛡️ Cibersecurity Journey

Este repositório é o meu laboratório central de Cibersegurança. Aqui, documento de forma estruturada minha evolução técnica, relatórios de auditoria, análises de logs e a implementação de ferramentas de defesa (Blue Team/SOC).

## 🚀 Trilhas de Estudo em Foco
* **Defesa Ativa (SOC):** Monitoramento, triagem de incidentes e análise de logs.
* **SecDevOps:** Integração de segurança no ciclo de vida de aplicações (ex: Pluck Manager).
* **Ofensiva (Futuro):** Estudos de Red Team e Bug Bounty via HackerOne e Hack The Box.
